# Project Documentation

This folder contains the baseline project documentation set for the public D&D campaign site.

## Documents
- `URS.md` - User Requirements Specification
- `DS.md` - Design Specification
- `FS.md` - Functional Specification
- `API.md` - API and data contract description
- `DEPLOYMENT.md` - GitHub Pages and custom domain setup
- `TESTING.md` - test approach and validation flow

## Scope
These documents describe the current site and build pipeline as implemented in this repository:
- static frontend in `docs/`
- build and sync scripts in `Tech/`
- public content only

Operational secrets, tokens, account details, and environment-specific security values are intentionally excluded.
